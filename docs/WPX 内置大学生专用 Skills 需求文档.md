# WPX 内置大学生专用 Skills 需求文档

**版本**：V1.0
**状态**：定稿
**关联文档**：PRD、内置教师 Skills 需求文档、用户中心（设置）

---

## 1. 概述

为大学生用户群体内置 16 款专用 Skills，覆盖学术写作、学习辅助、知识管理、展示汇报、学业规划五大场景。所有 Skills 随 WPX 安装即可用，无需登录，无需付费。

**设计原则**：
- 贴近大学生真实痛点，不堆砌功能
- 每个 Skill 必须有明确的使用场景和即时价值
- Prompt 模板基于高校教学实际，输出可直接使用
- 引导式交互，降低使用门槛

## 2. 内置 Skills 清单

### 2.1 学术写作类

**1. 论文大纲生成器** (`paper-outline`)
- **场景**：老师布置了一篇课程论文，不知道从哪开始
- **一句话触发**：“帮我生成一篇关于‘人工智能对劳动力市场的影响’的课程论文大纲”
- **能力**：自动生成包含摘要、引言、文献综述、分析框架、结论的标准学术大纲，每章节标注写作要点和推荐字数分配
- **输入**：论文主题、学科、字数要求、级别（本科/硕士）

**2. 文献综述助手** (`literature-review`)
- **场景**：面对一堆文献，不知道怎么梳理成综述
- **一句话触发**：“帮我把以下三篇论文的核心观点整合成一段文献综述”
- **能力**：分析输入的多篇文献摘要或笔记，自动梳理研究脉络、归纳流派、指出研究空白
- **输入**：粘贴文献摘要/笔记（可多段），指定综述维度

**3. 查重降重润色** (`paraphrase-academic`)
- **场景**：查重率太高，需要改写
- **一句话触发**：“这段文字查重没过，帮我在不改变原意的情况下改写”
- **能力**：保持学术严谨性的前提下，变换句式、替换词汇、重组段落结构，避免口语化
- **输入**：选中需要降重的文本

**4. 论文排版助手** (`paper-formatter`)
- **场景**：论文写完了，但格式要求一堆，目录、参考文献、页眉页脚调到头大
- **一句话触发**：“按本科毕业论文格式，帮我排版这篇论文”
- **能力**：根据学校模板自动生成封面、目录、页眉页脚、参考文献格式（支持 GB/T 7714、APA、MLA）
- **输入**：论文正文、学校名称、论文题目、作者信息

**5. 开题报告生成器** (`proposal-generator`)
- **场景**：要开题了，不知道开题报告怎么写
- **一句话触发**：“帮我写一份关于‘短视频对青少年心理健康影响研究’的开题报告”
- **能力**：生成包含选题背景、研究意义、国内外研究现状、研究内容与方法、进度安排、预期成果的完整开题报告框架
- **输入**：选题方向、学科、研究方法偏好

### 2.2 学习辅助类

**6. 笔记整理** (`note-organizer`)
- **场景**：上课记了一堆零散笔记，需要整理成结构化复习资料
- **一句话触发**：“帮我把这些课堂笔记整理成结构清晰的复习资料”
- **能力**：将散乱的笔记整理为分层级的知识点笔记，自动生成思维导图结构、标记重点概念
- **输入**：选中零散笔记文本

**7. 概念解释器** (`concept-explainer`)
- **场景**：教材上的定义看不懂
- **一句话触发**：“用通俗易懂的方式解释‘边际效用递减’，并举例说明”
- **能力**：将复杂概念用生活化语言解释，提供类比、案例、图解描述，可指定解释深度
- **输入**：概念名称、学科背景、解释深度（入门/进阶/考研）

**8. 错题复盘** (`mistake-review`)
- **场景**：刷了题但不知道错在哪
- **一句话触发**：“这道题我做错了，帮我分析错因，并总结这类题的通用解法”
- **能力**：分析错题，诊断知识盲点，总结通用解题模板，推荐同类练习方向
- **输入**：题目内容、自己的错误答案、正确答案

**9. 考前冲刺计划** (`exam-cram-plan`)
- **场景**：还有一周考试，不知道怎么高效复习
- **一句话触发**：“帮我制定一个7天高数期末复习计划，我基础不太好”
- **能力**：根据考试科目、剩余天数、当前水平，生成每日复习计划和重点优先级
- **输入**：科目、剩余天数、当前掌握程度、可用每日学习时间

---

### 2.3 知识管理类

**10. 知识库构建** (`knowledge-builder`)
- **场景**：想搭建一个学科知识体系，但知识点太碎
- **一句话触发**：“帮我搭一个‘西方经济学（微观部分）’的完整知识框架”
- **能力**：生成树状知识结构图（Markdown格式），标注各知识点的关联和依赖关系
- **输入**：学科/课程名称、教材版本、范围（全书/某章节）

**11. 复习卡片生成** (`flashcard-generator`)
- **场景**：需要背的东西太多，想要Anki风格的卡片
- **一句话触发**：“把以下知识点生成问答式复习卡片”
- **能力**：将输入的知识点拆解为“问题-答案”对，格式兼容主流记忆卡片应用
- **输入**：知识点文本、卡片数量

**12. 读书笔记辅助** (`reading-notes`)
- **场景**：读了一本书或一篇长文，想高效做笔记
- **一句话触发**：“帮我为以下内容生成读书笔记，包含核心观点、金句摘录和个人思考提示”
- **能力**：自动提取核心论点、关键论据，生成结构化笔记，预留个人思考区
- **输入**：选中文本或粘贴内容

---

### 2.4 展示汇报类

**13. 课堂汇报大纲** (`presentation-outline`)
- **场景**：小组作业要做PPT汇报，不知道讲什么、怎么讲
- **一句话触发**：“帮我为‘新能源汽车市场分析’设计一个10分钟的课堂汇报大纲”
- **能力**：生成包含开场、主体内容分段、数据展示建议、互动设计、总结的汇报大纲
- **输入**：主题、汇报时长、听众（老师/同学）、是否有数据

**14. 演讲稿生成** (`speech-writer`)
- **场景**：要上台演讲，稿子不会写
- **一句话触发**：“帮我写一篇5分钟的竞选学生会主席演讲稿”
- **能力**：根据不同场合（竞选、比赛、课程展示）生成风格匹配的演讲稿
- **输入**：演讲主题、场合、时长、个人亮点/经历

---

### 2.5 学业规划类

**15. 学业规划助手** (`academic-planner`)
- **场景**：大学四年不知道怎么规划
- **一句话触发**：“我是大二计算机专业的学生，帮我规划大学后三年的学习和实习安排”
- **能力**：根据专业、年级、目标（考研/就业/出国），生成分阶段规划建议
- **输入**：专业、年级、毕业后目标、当前绩点、已有经历

**16. 实习申请文书** (`internship-application`)
- **场景**：找实习，简历和求职信不会写
- **一句话触发**：“帮我写一份申请互联网运营实习的求职信，我有一段校园自媒体经历”
- **能力**：生成简历模板、求职信、自我介绍，根据用户经历定制亮点描述
- **输入**：目标岗位、个人经历、教育背景、申请公司

## 5. 与教师 Skills 的复用与区分

| 维度 | 教师 Skills | 大学生 Skills |
|:---|:---|:---|
| 写作类 | 教案、发言稿、通知 | 论文、开题报告、演讲稿 |
| 出题/做题类 | 出题、变式题 | 错题复盘、考前冲刺 |
| 批改/反馈类 | 作文批改、评语 | 查重降重、笔记整理 |
| 规划类 | 课题申报 | 学业规划、实习申请 |
| 共同点 | 都需要结构化输出、专业严谨 | 更强调学习引导、个人成长 |

- 教师和大学生 Skills **不重叠**，各自独立。
- 如果用户同时是教师和研究生，两个分类的 Skills 均可使用。

## 6. 技术实现

### 6.1 元数据存储

在 `src/data/built-in-skills.js` 中，大学生 Skills 与教师 Skills 存放在同一数组，通过 `category` 字段区分：
- `category: 'education'` → 教师专用
- `category: 'college'` → 大学生专用

### 6.2 加载逻辑

- WPX 启动时，内置 Skills 全部加载。
- Skills 管理页面通过筛选标签按 `category` 过滤展示。
- 每个 Skill 的 `promptTemplate` 和 `inputSchema` 与教师 Skills 格式完全一致。

### 6.3 调用方式

与教师 Skills 相同：
- 对话中自然语言触发，AI 自动匹配 Skill。
- 手动指定 Skill 名称优先匹配。
- 需要参数时弹出简易表单。
