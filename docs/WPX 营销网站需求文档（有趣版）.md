# WPX 营销网站需求文档（有趣版）

**项目代号**：WPX Land  
**目标**：让用户第一次看到官网就想下载，让WPS用户感到“终于有救了”，让学生和老师觉得“这个工具是偷窥了我的生活”。  
**调性**：**叛逆、幽默、极客、零套路**。我们是一个AI文档编辑器，但我们的网站绝不死板。

---

## 1. 核心定位与目标

- **一句话口号**（备选）：“写你所想，格式随你” / “AI文档伴侣，不绑架不恰饭” / “给你Word做不到的自由”。
- **目标**：驱动下载（桌面端）、建立品牌认知（与WPS形成鲜明对比）、传播“不付费也能用”的信任。
- **目标用户**：被WPS弹窗折磨的办公族、找免费工具的学生、需要AI辅助的教师、不想学复杂软件的创作者。

---

## 2. 整体风格与交互基调

- **视觉**：明亮极简，大面积留白，辅以活泼手绘插画（例如：一个AI小助手跳出纸张，打翻WPS的稻穗标志）。配色以WPX主题色（蓝紫渐变）为主，搭配黄色和薄荷绿强调。
- **动画**：
  - 流畅的微交互（按钮 hover 有弹性效果，卡片悬浮有景深）。
  - 背景有流动的代码/文字粒子（极客感）。
- **文案**：第一人称口语化，像朋友聊天，直接戳痛点。大量使用emoji。
- **彩蛋**：多个隐藏交互，例如：
  - 连续点击Logo七次会出现“开发者模式”小游戏（贪吃蛇吞掉WPS弹窗）。
  - 在任意页面按上上下下左右左右BA触发复古对话框。
  - 下载按钮的鼠标悬停文案随机变化：“别犹豫，又不要钱”、“给WPS最后一次机会？”、“Windows用户请猛击”、“Mac用户也有的”。

---

## 3. 网站结构（单页滚动为主，辅以少量子页）

### 3.1 导航栏（极简）
- 固定顶部，透明变白色。
- **Logo** + **功能** + **下载** + **关于**（或博客） + **GitHub**（开源有逼格）。
- 右侧醒目的【**免费下载**】按钮（带脉冲动画）。

### 3.2 首屏（Hero）
- **大标题**：“别再为WPS交税了。WPX，新一代AI文档编辑器。”
- **动态替换文字**：中间一行字循环变化：“Markdown写文？”“AI对话改稿？”“PDF转Word？”“表格？图片处理？”“还有免费字体和智能模板…”
- **视觉**：一个简短的演示动画（编辑器输入文字，AI头像右下角弹出，一句话修改，导出PDF）。
- **两个按钮**：【立即下载】(主) 【在线体验(Web版)】(次)。下载按钮旁小字：“仅15MB，不占C盘”。

### 3.3 痛点对比区（WPS受害者联盟）
- **标题**：“如果你也经历过这些，WPX是为你准备的。”
- 动画卡片轮播，左右分屏对比。
  - 左（WPS）：广告弹窗、PDF转Word要会员、模板要钱、安装包800MB。
  - 右（WPX）：纯静无广告、PDF转Word免费、AI一键生成模板、安装包15MB。
- 使用“撕纸”过渡动画，从WPS画面撕开露出WPX。

### 3.4 核心功能秀（动画演示）
- 不是死板的列表，而是一个可交互的“虚拟编辑器”演示区。
- 用户可以在网页上尝试简单的操作（例如选中一段文字，在弹出的模拟AI对话框里输入“翻译成英文”，然后看到一个假的流式输出效果）。不能真编辑，但体验真实。
- 旁边用文字介绍亮点：
  - **AI头像随时待命**：选中文字就能聊，像Cursor一样。
  - **图片轻处理**：去背景、打码一句话搞定。
  - **智能模板**：越用越懂你。
  - **16+教师技能、16+学生技能**：直接生成教案、论文大纲。
  - **开源字体随便用**：思源、霞鹜文楷已内置。

### 3.5 免费声明区（建立信任）
- **大字标题**：“为什么免费？”
- 说明商业模式：“工具永久免费，字体可自导入。只有你使用我们的云端AI算力或购买商店商业字体时才需要购买Token，而且6块钱起。其余一切不收费，不弹窗，不绑架。”
- 放一个极简的**收费对比表**，用显眼的颜色标出WPS的“按年订阅”和WPX的“按需毫毛Token”。

### 3.6 技能市场展示
- **标题**：“你的AI，你说了算。”
- 展示可自由启用的Skills网格，重点展示几个好玩的：“教案生成器”、“错题复盘”、“彩虹屁生成器”（如果放一个幽默技能会更有趣）。
- 有个“技能实验室”的互动小游戏：用户输入一句话，随机匹配一个技能并展示生成结果（娱乐性）。

### 3.7 用户评价（可选）
- 未来可加入，现在先放一些拟物化的“用户吐槽”：
  - “以前用WPS，现在用WPX，头发都长回来了。”—— 某不愿透露姓名的老师
  - “我把7zip那事儿忘干净了，直到它帮我解压了学生论文。”—— 在读研究生

### 3.8 下载引导区（重复）
- 再次出现【免费下载】按钮，并附上Windows/macOS/Linux图标。
- 小字：“无需注册即可使用大部分功能，用你自己的AI Key也行。”

### 3.9 页脚
- 简洁链接：GitHub、用户协议、隐私政策、关于我们。
- 一句俏皮话：“Made with ❤️ and a lot of midnight snacks.”

## 4. 特殊互动与彩蛋需求

1. **打字机效果**：首屏标语动态打字，模拟编辑器输入。
2. **鼠标拖尾**：可选开关，默认开启一串小✨。
3. **404页面**：一个有趣的“迷路”页面，展示：“你好像走丢了，但别担心，WPX的AI能帮你找回来。”附带一个AI对话小窗口，可输入搜索。
4. **下载确认彩蛋**：用户点击下载后，按钮变“正在召唤魔法阵...”，然后正常下载。
5. **对比模式**：痛点对比区可以拖动一根中线，实时切换WPS和WPX的画面（交互滑块）。

## 5. 技术与性能要求

- 纯静态网站，使用Vue 3 + Vite + Tailwind CSS构建，便于集成和部署。
- 动画库使用 **GSAP** 或 **Motion One** 实现高性能动画。
- 所有文案、对比数据通过JSON配置，方便后续A/B测试。
- SEO友好，使用Vite的SSG插件预渲染静态页面。
- 部署在 `wpx.proclaw.cc`，全球CDN加速。
- 埋点统计下载量、功能区交互（后续迭代）。

4. **制作痛点对比滑块组件**。
5. **构建虚拟编辑器互动区（模拟AI对话）**。
6. **实现技能网格和随机匹配游戏**。
7. **添加彩蛋（Konami Code、Logo点击计数）**。
8. **实现免费声明区和页脚**。
9. **添加响应式适配，移动端优化**。
