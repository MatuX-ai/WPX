1. 概述
基于 CopilotKit 技术，为 WPX 新增 AI 驱动的 HTML5 演示文稿生成能力。用户通过自然语言与 AI 助手进行四步交互，即可在编辑器内直接生成可翻页、可交互、可修改的 HTML5 幻灯片，并支持导出为标准 PPTX 或独立可交互网页。

核心价值：让 PPT 制作从“手动拖拽”变为“对话生成”，从“静态文件”变为“活的组件”。

2. 用户交互流程（四步法）
第一步：生成大纲 → 编辑器展示，用户可修改
用户对 AI 说：“做个关于新能源汽车的发布会 PPT”。

AI 调用内置的“PPT 大纲生成”能力，在编辑器中生成结构化 Markdown 大纲。

大纲格式：

text
# 新能源汽车发布会
## 封面：新能源汽车的未来已来
## 市场背景：全球新能源市场概览
## 核心优势：三大技术突破
## 产品矩阵：覆盖全场景
## 未来展望：2026-2030 规划
## 结束页：谢谢
AI 在对话窗中询问：“大纲已生成，请直接在编辑器中修改，确认后告诉我。”

用户可直接在编辑器中编辑大纲（增删改页面、调整标题、补充要点）。

修改完成后，用户说“确认”或“继续”，进入下一步。

第二步：AI 询问模板偏好 → 用户选择或描述
AI 在对话窗中展示模板选项（卡片形式）：

🏢 商务简约风：白底蓝调，适合正式场合

🚀 科技感风：深色背景，发光元素，适合发布会

🌿 清新自然风：浅绿配色，适合教育/公益

✨ 自定义：请描述你想要的风格

用户可以：

点击选择一个模板卡片

或者说：“我要特斯拉发布会那种黑底白字的极简风”

AI 确认模板后，进入下一步。

第三步：AI 一张张搭建 HTML5 幻灯片
AI 在对话窗中提示：“正在生成幻灯片，请稍候...”

AI 逐页生成幻灯片，编辑器中央展示一个可翻页的 HTML5 幻灯片预览。

每一页都是真实的组件渲染：

封面页：标题 + 副标题 + 背景图

文字页：标题 + 要点列表

图表页：标题 + 柱状图/折线图（用 ECharts 渲染）

图文页：左侧文字 + 右侧图片

结束页：感谢语

生成完成后，AI 提示：“幻灯片已生成，共 X 页。你可以直接修改任意页面，或对我说你的修改想法。”

第四步：自然语言修改 + 转换导出
用户可通过自然语言继续修改：

“把第三页的图表换成柱状图”

“整体色调改成蓝色”

“加一页关于竞品分析的，放在第四页之后”

“封面标题字体再大一点”

用户说“转换为可交互网页”，AI 将整套幻灯片打包为一个独立的 HTML 文件（内嵌翻页动画、图表交互、导航菜单）。

用户说“导出为 PPTX”，系统调用 PptxGenJS 将幻灯片数据转为标准 PPTX 文件下载。

用户说“导出为 PDF”，系统将每页幻灯片渲染为 PDF 页面。

3. 技术方案
3.1 幻灯片组件库
在 WPX 中创建一套 HTML5 幻灯片组件，每个组件负责一种页面类型：

组件名	用途	关键 props
CoverSlide	封面页	title, subtitle, backgroundImage, theme
TocSlide	目录页	items（标题列表）
TextSlide	纯文字页	title, bulletPoints（要点数组）
ImageTextSlide	图文页	title, text, imageUrl, layout（左图右文/右图左文）
ChartSlide	图表页	title, chartType, chartData（ECharts 配置）
TableSlide	表格页	title, tableData（行列数据）
EndSlide	结束页	text, contactInfo
3.2 CopilotKit 集成
受控模式：将上述组件注册到 CopilotKit 的 Agent 中，Agent 负责选择合适的组件并填入数据。

A2UI：Agent 输出的 JSON 格式描述每一页用哪个组件、填充什么数据。

双向同步：用户在编辑器里修改幻灯片内容，Agent 能感知并调整后续生成。

3.3 幻灯片预览与编辑
使用 reveal.js 或自定义 Vue 组件实现幻灯片预览（支持翻页、全屏）。

预览嵌入在 Tiptap 编辑器的一个特殊节点中（SlideDeckNode）。

用户点击幻灯片页面可直接编辑文字、替换图片。

3.4 导出能力
HTML 网页：将幻灯片组件打包为独立 HTML 文件，包含 reveal.js 或自定义翻页逻辑。

PPTX：使用 PptxGenJS 将幻灯片数据转为标准格式。

PDF：使用 Puppeteer 或浏览器打印 API 将 HTML 幻灯片渲染为 PDF。

4. 与现有 WPX 功能的关系
现有功能	如何联动

字体库	幻灯片使用 WPX 内置免费字体，支持切换
