# WPX 虚拟纸张与导出母版需求文档

**版本**：V1.0  
**状态**：定稿  
**关联文档**：PRD、UI/UX设计规范、导出功能需求

---

## 1. 概述

为 WPX 引入传统的“纸张大小”概念，但以非侵入式的方式实现。编辑器保持无限画布的自由度，纸张大小仅作为“导出母版”和可选的“焦点写作模式”存在。用户写作时不受纸张约束，导出时由 AI 自动按预设母版排版。

**核心原则**：编辑无纸张，导出有边界。

---

## 2. 功能模块

### 2.1 默认纸张设置

在设置页的“通用”板块中，用户可设定默认导出纸张。

| 设置项 | 选项 | 默认值 |
|:---|:---|:---|
| **默认纸张** | A4 / Letter / 16开 / 手机长图(375px宽) / 无 | A4 |
| **默认页边距** | 宽(25mm) / 标准(20mm) / 窄(15mm) / 自定义 | 标准 |
| **页眉页脚** | 无 / 仅页码 / 自定义文字 | 无 |

- “无”模式：导出时不限制页面宽度，适合纯网页/移动端阅读。
- 修改后不影响编辑器，仅影响导出结果和焦点模式的参考线宽度。

---

### 2.2 焦点写作模式（可选开启）

用户在编辑过程中，可选择开启“纸张参考线”，获得类似 Word 页面视图的仪式感，但无分页符限制。

| 特性 | 说明 |
|:---|:---|
| **开启方式** | 编辑器右上角切换按钮：⬜ 显示纸张 / ䷮ 无边画布 |
| **视觉效果** | 编辑区中央出现白色竖线框（模拟纸张宽度），背景变为浅灰色(#F0F0F0)，产生悬浮纸张的视觉层次 |
| **内容约束** | 文字自动在框内换行，但图片、表格、代码块可超出框宽，不会被裁剪 |
| **无分页符** | 内容连续滚动，不显示任何分页线，用户无需关心断页位置 |
| **宽度参考** | 底部状态栏显示当前纸张宽度（如“A4 · 210mm”），仅作信息提示 |
| **记忆** | 开启/关闭状态全局记忆，下次启动保持 |

**纸张宽度像素映射**（基于 96dpi）：
| 纸张 | 宽度 |
|:---|:---|
| A4 | 794px |
| Letter | 816px |
| 16开 | 728px |
| 手机长图 | 375px |

### 2.3 AI 排版引擎（导出时触发）

导出 Word/PDF 时，WPX 基于当前 Markdown 内容 + 预设纸张母版，由 AI 自动完成排版。

| 步骤 | 说明 |
|:---|:---|
| **1. 内容分析** | AI 扫描 Markdown 结构：标题层级、段落、图片尺寸、表格列数 |
| **2. 分页计算** | 根据纸张高度和页边距，计算每页可容纳的行数，自动插入分页符 |
| **3. 图片适配** | 超出页面宽度的图片自动缩放至适配宽度；小图片保留原尺寸 |
| **4. 表格处理** | 宽表格自动调整列宽或转为横向页面 |
| **5. 页眉页脚** | 根据设定生成页码、文档标题等 |
| **6. 输出预览** | 导出前展示分页预览效果（可选查看） |

**导出前确认框**：
```
┌──────────────────────────────────────┐
│  导出设置                             │
│                                      │
│  纸张：[A4 ▾]  页边距：[标准 ▾]        │
│                                      │
│  ☑ 自动分页                          │
│  ☑ 图片适配页面宽度                   │
│  ☐ 生成目录（若有3个以上标题）         │
│                                      │
│  [查看预览]  [确认导出]               │
└──────────────────────────────────────┘
```

### 2.4 快捷入口（非侵入式）

在编辑器右下角 AI 头像旁边，提供快捷切换导出母版的小入口。

| 元素 | 说明 |
|:---|:---|
| **显示文字** | 半透明小字：“导出母版：A4” |
| **点击行为** | 弹出简短的纸张选择菜单：A4 / Letter / 16开 / 手机长图 / 无 |
| **选中后** | 文字即时更新，Toast 提示“导出母版已切换为 A4” |
| **可见性** | 始终可见，但半透明不干扰；焦点模式下更加显眼 |

---

### 2.5 与无限画布的共存逻辑

| 场景 | 行为 |
|:---|:---|
| **日常写作** | 无纸张参考线，无限画布，自由输入 |
| **开启焦点模式** | 显示纸张竖线框，文字在框内但不强制 |
| **插入超大元素** | 图片/表格可超出参考线框，编辑区不裁剪 |
| **导出** | 自动按预设纸张分页、缩放、排版 |
| **切换纸张** | 不影响已有内容，仅改变导出预期和参考线宽度 |

## 3. 技术实现要点

### 3.1 焦点模式的 CSS 实现

- 编辑区容器始终 100% 宽度。
- 焦点模式下，给 Tiptap 的 `.ProseMirror` 添加 `max-width: 794px; margin: 0 auto;`。
- 背景色通过父容器切换 class 实现。
- 不修改任何编辑器节点结构。

### 3.2 导出时 AI 排版

- 用户触发导出 → 后端调用 Pandoc 或直接拼装 docx/PDF。
- 在转换前，将纸张大小、页边距等参数传入。
- 对于 PDF，Pandoc 支持 `--pdf-engine-opt=-V --pdf-engine-opt=geometry:a4paper` 等参数。
- 对于 docx，通过参考模板（reference.docx）设置页面尺寸。
- 未来可加入 AI 分页优化：检测段落、图片位置，避免孤行、改善排版。

## 4. UI 原型（简要）

### 焦点模式按钮
```
┌────────────────────────────────────┐
│  [≡] [⬜/䷮] 文档名.md     [⚙️] [👤] │  ← 标题栏右上角
└────────────────────────────────────┘
```
点击 ⬜ 切换焦点模式。

### 右下角母版入口
```
                    ┌──────────────┐
                    │ 🤖 AI 助手    │
                    └──────────────┘
                    导出母版：A4 ▼
```
半透明，不遮挡操作。

## 5. 验收标准

1. 设置页“默认纸张”可切换 A4/Letter/16开/手机长图/无。 ✅
2. 焦点模式下，编辑器中央显示纸张宽度的白色区域，背景灰色。 ✅
3. 焦点模式下，图片和表格可超出参考线框。 ✅
4. 右下角显示当前导出母版，点击可快速切换。 ✅
5. 导出 PDF/Word 时，按预设纸张分页和排版。 ✅
6. 导出确认框可临时切换纸张大小。 ✅
7. 默认纸张为“A4”，用户不设任何参数也能导出正常文档。 ✅
