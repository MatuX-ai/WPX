# WPX 管理后台需求文档

**版本**：V1.0
**状态**：定稿
**关联文档**：PRD、用户认证、字体库、Skills、Token收费

---

## 1. 概述

WPX 管理后台（admin.proclaw.cc）是 WPX 产品的运营管理中枢。管理员在此监控用户数据、管理字体商店、配置 AI 模型、查看 Token 消费、管理 Skills 上下架、发布公告等。

**核心原则**：简洁高效，数据可视化优先，所有操作可追溯。

---

## 2. 整体结构

### 2.1 侧边栏导航

```
admin.proclaw.cc
├── 📊 仪表盘
├── 👥 用户管理
├── 🤖 AI 模型配置
├── 🔤 字体商店管理
├── ⚡ Skills 管理
├── 💰 Token 与订单
├── 📢 公告与版本
├── ⚙️ 系统设置
└── 📋 操作日志
```

### 2.2 登录与权限

- 管理员通过 `account.proclaw.cc` 统一认证登录。
- 角色分为：超级管理员、运营管理员、内容编辑。
- 权限控制：
  - 超级管理员：全部功能 + 系统设置。
  - 运营管理员：用户管理、Token/订单、公告、字体审核。
  - 内容编辑：Skills 管理、字体商店内容更新。

## 3. 功能模块详述

### 3.1 仪表盘

**作用**：一眼看清 WPX 整体运营状况。

**展示卡片**：
- 今日活跃用户数（DAU）
- 今日新增注册用户
- 今日 AI 调用次数（免费/付费）
- 今日 Token 充值总额
- 今日字体导出次数（按字体统计 Top 5）
- 活跃 Skills Top 10
- 7 日用户增长趋势图（折线图）
- 7 日 AI 调用趋势图（免费 vs 付费，堆叠柱状图）

---

### 3.2 用户管理

**用户列表**
- 字段：用户 ID、昵称、邮箱、注册时间、最后活跃时间、状态（正常/禁用）。
- 搜索：支持按邮箱、昵称模糊搜索。
- 筛选：注册时间段、状态。
- 操作：查看详情、禁用/启用、删除。

**用户详情**
- 基本信息：昵称、邮箱、头像、注册时间。
- 使用统计：总活跃天数、总 AI 调用次数、总文档数。
- Token 余额与消费记录。
- 已购字体列表。
- 已启用 Skills 列表。

**访客统计**
- 访客设备 ID 数量
- 访客 AI 调用次数（免费额度使用情况）
- 访客活跃趋势

---

### 3.3 AI 模型配置

**公共模型管理**
- 当前提供的免费公共模型列表（如 DeepSeek-V3）。
- 每款模型的配置：
  - 模型名称、API Endpoint、API Key。
  - 每日免费调用次数上限（默认 50 次/设备）。
  - 状态（启用/停用）。
- 支持添加新模型、编辑、停用。

**调用监控**
- 实时显示当前 QPS（每秒请求数）。
- 今日调用总量、成功/失败率。
- 按模型分类的调用量统计。
- 错误日志：最近 100 条调用失败记录（时间、用户、错误信息）。

---

### 3.4 字体商店管理

**字体库管理**
- 字体列表：名称、厂商、类型、价格（Token/字）、状态（上架/下架）。
- 搜索、筛选：按类型（黑体/宋体/手写/装饰）、厂商、状态。
- 操作：添加新字体、编辑、上架/下架、删除。

**添加/编辑字体**
- 字体名称、厂商、类型、标签。
- 预览图上传。
- 字体文件上传（.ttf/.otf）。
- 定价：Token/字。
- 描述文字。

**使用统计**
- 每款字体的累计导出次数。
- 每款字体的累计 Token 消耗。
- 7 日/30 日使用趋势。
- 用户 Top 10（使用该字体最多的用户）。

---

### 3.5 Skills 管理

**内置 Skills 列表**
- 展示所有内置 Skills（教师 16 款 + 学生 16 款 + 通用 N 款）。
- 字段：ID、名称、分类、状态（启用/停用）。
- 操作：启用/停用。

**在线 Skills 管理**
- 管理 skillhub.proclaw.cc 上的 Skills。
- 添加新 Skill：名称、分类、描述、图标、Prompt 模板、输入字段定义。
- 编辑已有 Skill。
- 上架/下架。
- Skills 使用统计：被调用次数、活跃用户数。

**社区 Skills 审核**（未来）
- 审核用户提交的 Skills，通过后上架到 skillhub。

---

### 3.6 Token 与订单

**充值订单管理**
- 订单列表：订单号、用户、金额、Token 数量、支付方式、时间、状态。
- 搜索：订单号、用户邮箱。
- 筛选：时间段、状态（待支付/已支付/已退款）。
- 支持手动退款（全额）。

**消费记录**
- 记录列表：用户、字体、字数、Token 消耗、文档哈希、时间。
- 统计：按天/周/月的消费趋势图。
- 按字体统计消费排行。

**收入统计**
- 今日/本周/本月/总计收入。
- 收入趋势图（折线图）。
- 按充值包统计（10元包 vs 自定义充值）。

---

### 3.7 公告与版本

**应用公告**
- 公告列表：标题、内容摘要、发布状态、发布时间。
- 添加公告：标题、正文（Markdown）、生效时间、过期时间。
- 公告展示位置：WPX 启动时弹窗、设置页面顶部横幅。

**版本管理**
- 版本记录：版本号、发布日期、更新内容（Markdown）、下载链接。
- 添加新版本：版本号、更新日志、Windows/macOS/Linux 下载链接。
- 版本更新检查：WPX 客户端定期请求最新版本信息，后台控制是否强制更新。

---

### 3.8 系统设置

**基础配置**
- 应用名称（WPX）。
- 免费 AI 调用次数上限（默认 50 次/天/设备）。
- 最大窗口数限制（默认 8 个）。
- 是否开放注册开关。

**CDN 与存储配置**
- 字体文件 CDN 地址。
- Skills 文件 CDN 地址。
- 安装包下载 CDN 地址。

**管理员账号**
- 管理员列表。
- 添加/编辑/删除管理员账号。
- 角色分配。

---

### 3.9 操作日志

- 记录所有管理员操作：
  - 时间、操作人、操作类型、操作详情（JSON 格式）。
- 支持按操作人、操作类型、时间筛选。
- 日志保留 90 天，支持导出 CSV。

---

## 4. 技术要点

- **前端**：Vue3 + Vite + Tailwind CSS + ECharts（图表）。
- **后端**：Node.js 或 Go，RESTful API。
- **数据库**：PostgreSQL。
- **认证**：对接 `account.proclaw.cc` 的 JWT 认证，管理员角色校验。
- **文件存储**：字体文件、预览图等使用对象存储（OSS/S3）+ CDN。
- **实时数据**：仪表盘数据使用 Redis 缓存，定期刷新。
