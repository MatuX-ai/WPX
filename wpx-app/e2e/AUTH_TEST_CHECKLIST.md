## 手动测试步骤（Electron 完整验收）

### 1. 首次启动为访客，TitleBar 显示「登录」

**前置**：清除凭据（删除用户数据目录中的 auth / free-quota store，或全新安装）。

**步骤**：
1. 启动 WPX：`npm run electron:dev`
2. 等待启动加载完成

**预期**：
- TitleBar 右侧显示 **「登录」** 按钮
- 无用户头像 / 昵称

---

### 2. 访客可使用编辑器；AI / 模型设置显示免费模型与剩余次数

**步骤**：
1. 新建文档，输入文字，确认可正常编辑
2. 打开 **设置 → 模型配置**
3. 确认遮罩显示 **「登录后解锁」**，下层可见免费模型与剩余次数
4. （可选）打开 AI 写作助手，确认使用平台免费模型

**预期**：
- 编辑器功能正常
- 展示 **WPX 免费模型** 与 **今日剩余调用次数**（如 `剩余 x/50 次`）

---

### 3. 免费次数用尽后，AI 返回提示

**前置**：将访客当日免费次数消耗至 0。

**步骤**：
1. 打开 AI 对话窗
2. 发送一条消息

**预期**：
- 显示：**「今日免费次数已用完，请明天再试或登录获取更多次数」**
- 提供 **「登录」** 引导按钮

---

### 4. 点击登录，浏览器跳转 account.proclaw.cc；回调后已登录

**步骤**：
1. 点击 TitleBar **「登录」**
2. 确认浏览器打开 `https://account.proclaw.cc/...`
3. 完成 account 登录
4. 通过 `wpx://auth?...` 回调客户端

**预期**：
- 回调成功后 TitleBar 变为已登录态

---

### 5. 登录后 TitleBar 显示头像和昵称；可退出

**步骤**：
1. 点击用户区域 → **「退出登录」**

**预期**：
- 菜单含 **「我的设置」**、**「退出登录」**
- 退出后恢复 **「登录」** 按钮

---

### 6. 登录后可添加自定义 API Key 并测试连接

**步骤**：
1. **设置 → 模型配置**（无遮罩）
2. 选择 **「使用自定义模型」**，填写 Endpoint / API Key / 模型名
3. 点击 **「测试连接」**

**预期**：Toast 提示 **连接成功**（或明确失败原因）

---

### 7. 登录后可充值 Token 并使用商业字体导出

**步骤**：
1. 访问 **Token 充值** 页（非访客引导页）
2. 充值或模拟支付
3. 编辑器应用商业字体 ⚡，**导出 PDF**
4. 扣费确认框 **确认导出**

**预期**：扣费成功并完成导出

---

### 8. 退出登录后恢复访客；自定义模型与 Token 不可用

**步骤**：
1. 退出登录
2. 打开 **模型配置**、**字体与 Token**

**预期**：
- **「登录后解锁」** 遮罩
- 无法使用充值 / 自定义模型完整配置

---

### 9. 应用重启后，已登录状态保持（JWT 未过期）

**步骤**：
1. 登录后完全退出并重启 WPX

**预期**：仍为已登录，无需重新浏览器登录

---

### 10. JWT 过期且 refresh 失败，恢复访客模式

**前置**：refresh token 无效或服务端 401。

**步骤**：
1. 重启应用触发 `restoreSession()`

**预期**：凭据清除，恢复访客 **「登录」** 按钮
